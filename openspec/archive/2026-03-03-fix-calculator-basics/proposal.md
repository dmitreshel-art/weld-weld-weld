# fix-calculator-basics

## Intent

Сделать калькулятор функциональным, добавив недостающие поля ввода и подключив их к формулам расчёта.

На данный момент калькулятор НЕ РАБОТАЕТ:
- 12 полей ввода отсутствуют в HTML
- 13 переменных неинициализированы (undefined)
- exel.json не загружается
- Все расчёты возвращают NaN

## Scope

### In Scope

- [ ] Добавить недостающие input поля для параметров шва
- [ ] Подключить переменные в formulas.js к UI элементам
- [ ] Загрузка параметров из exel.json при выборе соединения
- [ ] Базовая обработка undefined/NaN значений
- [ ] Отображение результатов расчёта

### Out of Scope

- ❌ Валидация min/max значений (отдельный change)
- ❌ Рефакторинг duplicate code (отдельный change)
- ❌ Динамическое скрытие/показ полей по типу соединения
- ❌ Красивый UI/UX для новых полей
- ❌ Тесты

## Success Criteria

1. **Given** выбрано соединение С1 (ГОСТ 5264, толщина 1-4мм)
   **When** введены все параметры
   **Then** отображается Fn ≠ NaN, Mn ≠ NaN, Ge ≠ NaN, He ≠ NaN

2. **Given** выбрано соединение С8 (V-образное)
   **When** введены параметры включая углы
   **Then** расчёт выполняется корректно

3. **Given** выбрано трубное соединение
   **When** введён диаметр
   **Then** He рассчитывается как Ge × d × π

4. **Given** параметры не введены (undefined)
   **When** нажата кнопка расчёта
   **Then** показывается сообщение "Введите все параметры"
