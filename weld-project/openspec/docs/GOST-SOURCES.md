## Обозначения параметров

| Символ | Название | Единица | Описание |
|--------|----------|---------|----------|
| **S** | Толщина металла | мм | Толщина свариваемых деталей |
| **b** | Зазор | мм | Расстояние между кромками |
| **e** | Ширина шва | мм | Ширина усиления шва |
| **g** | Высота усиления | мм | Высота наплавленного металла |
| **c** | Притупление | мм | Нескошенная часть кромки |
| **α** | Угол скоса кромки | ° | Угол разделки верхней кромки |
| **β** | Угол скоса кромки | ° | Угол разделки нижней кромки |
| **R** | Радиус кривизны | мм | Радиус галтели |
| **k** | Катет шва | мм | Длина катета углового шва **(для У1, У2, Т1, Н1, Н2)** |
| **n** | Выступ конца | мм | Выступающая часть детали |
| **B** | Ширина нахлёстки | мм | Ширина перекрытия деталей **(для Н1, Н2)** |

## Формулы расчёта

### Площадь сечения (Fn)

| Тип соединения | Формула | Примечание |
|----------------|---------|------------|
| С1, С3 | Fn = e × S | Без зазора |
| С2, С4 | Fn = e × g + b × S | С зазором |
| С8, С17 (V-образное) | Fn = e × g + b × S + (S - c)² × tg(α/2) | С углом |
| **У1 (угловое)** | **Fn = 0.5 × k²** | **k — катет шва** |
| У2 (криволинейный) | Fn = e × g + R × g | С радиусом |
| **Т1 (тавровое)** | **Fn = k × S + 0.5 × k²** | **С учётом толщины** |
| **Н1 (нахлёсточное)** | **Fn = 0.5 × k²** | **Один угловой шов** |
| **Н2 (нахлёсточное)** | **Fn = k²** | **Два угловых шва** |

### Масса наплавленного металла (Mn)

```
Mn = ρ × Fn (г/м)
где ρ = 0.00785 г/мм³ (плотность стали)
```

### Удельная норма расхода (Ge)

```
Ge = k × Mn (г/м)
где k — коэффициент потерь:
  MMA (РД):  1.70
  MIG/MAG:  1.15
  TIG (РАД): 1.10
  SAW (АФ):  1.02
```

### Норма расхода на стык (He)

```
He = Ge × Lh (г)
где Lh — длина шва:
  - Плоское соединение: Lh = 1 м
  - Труба: Lh = d × π (где d — диаметр)
```
