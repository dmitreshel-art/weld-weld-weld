## Interaction Flow

```
User Opens App
      │
      ▼
┌─────────────────────┐
│ Show Empty State    │
│ "Выберите метод..." │
└─────────────────────┘
      │
      ▼
User Selects Method (e.g., MMA)
      │
      ▼
┌─────────────────────┐
│ Filter ГОСТ options │
│ Show ГОСТ dropdown  │
└─────────────────────┘
      │
      ▼
User Selects ГОСТ (e.g., ГОСТ 5264)
      │
      ▼
┌─────────────────────┐
│ Filter Joint Types  │
│ Show Joint list     │
└─────────────────────┘
      │
      ▼
User Selects Joint (e.g., С2)
      │
      ▼
┌─────────────────────┐
│ Load Joint Data     │
│ Show Diagram        │
│ Show Parameters     │
│ Populate Defaults   │
└─────────────────────┘
      │
      ▼
User Enters Parameters
      │
      ▼
┌─────────────────────┐
│ Validate Inputs     │
│ If valid: Calculate │
│ If invalid: Error   │
└─────────────────────┘
      │
      ▼
┌─────────────────────┐
│ Display Results     │
│ Fn, Mn, Ge, He      │
└─────────────────────┘
      │
      ▼
User Changes Theme
      │
      ▼
┌─────────────────────┐
│ Toggle Dark/Light   │
│ Save to LocalStorage│
└─────────────────────┘
```

## Accessibility Considerations

### Keyboard Navigation
```
Tab order:
1. Method radio buttons
2. ГОСТ dropdown
3. Joint type list
4. Parameter inputs (in order)
5. Theme toggle
6. Results area
```

### ARIA Labels
```html
<nav aria-label="Выбор параметров сварки">
  <fieldset aria-labelledby="method-legend">
    <legend id="method-legend">Метод сварки</legend>
    <input type="radio" aria-describedby="method-hint" />
  </fieldset>
</nav>

<main aria-label="Результаты расчёта">
  <output aria-live="polite" aria-atomic="true">
    Fn = 4.5 мм²
  </output>
</main>
```

### Focus Indicators
```css
/* Visible focus ring for keyboard users */
input:focus-visible,

  outline: 2px solid var(--primary);
  outline-offset: 2px;
```
