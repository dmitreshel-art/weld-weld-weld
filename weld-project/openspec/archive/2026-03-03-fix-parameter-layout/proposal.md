# fix-parameter-layout

## Intent
Исправить layout добавленных параметров — поля не влезают и съезжают вниз.

## Problem
После добавления 13 input полей в `parameterInputs`, они используют класс `userform__select-i` с фиксированной шириной 400px, что ломает layout.

## Scope

### In Scope
- [ ] Уменьшить ширину input для параметров (200px вместо 400px)
- [ ] Уменьшить размер label
- [ ] Добавить CSS для параметров в сетку (2 колонки)
- [ ] Скрыть неиспользуемые параметры по умолчанию

### Out of Scope
- ❌ Редизайн всего UI
- ❌ Адаптивный дизайн для мобильных

## Success Criteria
- Все input поля влезают в layout
- Нет горизонтального скролла
- Визуально аккуратный вид

## Estimated Effort
- 2-3 задачи
- ~15-30 минут
