# improve-json-population

## Intent
Улучшить загрузку и автоматическое заполнение параметров из exel.json при выборе типа соединения.

**Текущая проблема:** JSON загружается, но поля не заполняются автоматически при выборе типа/толщины. Пользователь вынужден вводить параметры вручную.

## Scope

### In Scope
- [ ] Автозаполнение параметров при выборе типа соединения
- [ ] Автозаполнение параметров при выборе толщины
- [ ] Показ применимых параметров для типа соединения
- [ ] Скрытие неприменимых параметров

### Out of Scope
- ❌ Добавление новых данных в JSON
- ❌ Редактирование JSON пользователем
- ❌ Валидация данных из JSON

## Success Criteria

1. **Given** выбрано соединение С1 и толщина 2
   **When** данные есть в JSON
   **Then** поля gap, weldWidth, gainHeight заполняются автоматически

2. **Given** выбрано соединение С8
   **When** у соединения есть параметр angleAlpha
   **Then** поле angleAlpha становится видимым

3. **Given** выбрано трубное соединение
   **When** тип соединения требует диаметр
   **Then** поле diameter становится видимым

## Affected Capabilities

| Capability | Change Type | Description |
|------------|-------------|-------------|
| `joint-geometry` | MODIFIED | Автозаполнение параметров |
| `user-interface` | MODIFIED | Показ/скрытие параметров |

## Risks & Mitigations

| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| Данные в JSON неполные | High | Medium | Fallback на пустые значения |
| Параметры не совпадают с типом | Medium | Low | Показывать только применимые |
| JSON структура изменилась | Low | High | Валидация структуры |
